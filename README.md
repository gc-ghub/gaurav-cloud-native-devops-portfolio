# gaurav-cloud-native-devops-portfolio
A complete Cloud-Native DevOps Portfolio project built on Azure AKS using Kubernetes, Helm, Istio Service Mesh, ArgoCD GitOps, and GitHub Actions CI/CD pipelines. Includes multi-environment deployments (dev, stage, prod), canary releases, traffic splitting, observability with Kiali/Grafana/Prometheus, and versioned microservices (v1/v2/v3).
